# 画面設計（MVP）

## 画面一覧
1. ダッシュボード（プロジェクト一覧）
2. プロジェクト詳細（最近の決定/タスク/議事録）
3. 意思決定一覧（検索・タグ）
4. 意思決定詳細（選択肢/基準/理由/リンク）
5. タスクボード（ToDo/Doing/Done）
6. 議事録一覧
7. 議事録詳細（決定・タスクへのリンク）

## 各画面の最低要件

### 1. ダッシュボード
- プロジェクト一覧表示
- 新規プロジェクト作成

### 2. プロジェクト詳細
- 最近の意思決定（最新5件）
- 期限が近いタスク（最新5件）
- 最近の議事録（最新5件）

### 3. 意思決定一覧
- キーワード検索
- タグ絞り込み

### 4. 意思決定詳細
- 決定タイトル / 決定日 / 決定者
- 選択肢A/B/C（複数行でもOK）
- 判断基準（複数）
- 決定理由
- 関連タスク / 関連議事録

### 5. タスクボード
- ToDo/Doing/Done の列
- タスクの移動（最初は状態変更ボタンでもOK）

### 6. 議事録一覧
- 日付順に表示
- 新規作成

### 7. 議事録詳細
- アジェンダ
- 決定事項
- 未決事項
- ToDo（タスクへのリンク）
- 関連意思決定のリンク
